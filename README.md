Колёса Свободы – Мобильное приложение аренды автомобилей
Мобильное приложение компании «Колёса Свободы», созданное на основе ранее разработанного веб-сайта автопарка.
Проект выполнен в рамках учебной работы по разработке программного обеспечения.

Описание проекта
Приложение предназначено для бронирования автомобилей через мобильное устройство.
Основой послужил сайт автопарка, который был переработан: серверная часть на Python была удалена, и вся логика реализована с использованием JavaScript.
Приложение работает по клиент-серверному принципу и взаимодействует с базой данных.

Основной функционал
Для пользователя:
Регистрация
Авторизация
Просмотр каталога автомобилей
Просмотр информации о выбранном автомобиле
Бронирование автомобиля
Просмотр своих заявок

Для администратора:
Просмотр всех бронирований
Подтверждение или отклонение заявок
Управление списком автомобилей

Используемые технологии
JavaScript
HTML
CSS
MySQL (база данных)
REST API (для обмена данными)
Технология мобильной разработки

Структура проекта
Веб/серверная часть:
├── static/
│   ├── css/
│   └── js/
├── templates/
└── server.js (или основной JS-файл)
Мобильное приложение:
├── screens/      # Экраны приложения
├── components/   # Компоненты интерфейса
├── services/     # Работа с API
└── main.(расширение)

База данных
В проекте используются следующие таблицы:
clients – клиенты
cars – автомобили
bookings – бронирования
trips – поездки

Принцип работы
Пользователь выполняет действие в мобильном приложении.
Приложение отправляет запрос к серверу.
Сервер обрабатывает запрос с помощью JavaScript.
Данные сохраняются или получаются из базы MySQL.
Результат возвращается в приложение и отображается пользователю.

Запуск проекта
Запуск серверной части
Установить зависимости:
npm install
Запустить сервер:
node server.js
Запуск мобильного приложения
Запуск зависит от выбранной технологии разработки.
Например:
npm start
или запуск через Android Studio.

Цель проекта
Разработка мобильного приложения на основе существующего веб-сайта автопарка с использованием JavaScript и клиент-серверной архитектуры.

Проект демонстрирует:
работу с базой данных
авторизацию пользователей
обработку заявок
взаимодействие мобильного приложения с сервером
