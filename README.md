# 🎮 Tic-Tac-Toe Mobile App

> Учебный проект по разработке мобильного приложения  
> Классическая игра «Крестики-нолики» на базе веб-технологий

---

## 📌 О проекте

Данный проект представляет собой мобильное приложение — реализацию классической игры **«Крестики-нолики»**, разработанную с использованием веб-технологий и адаптированную под мобильные устройства с помощью **Apache Cordova**.

Основная идея проекта — создать простое, удобное и понятное приложение, позволяющее играть в одну из самых известных логических игр прямо на смартфоне.

Несмотря на простоту концепции, проект охватывает важные аспекты разработки:
- построение интерфейса  
- обработка действий пользователя  
- реализация игровой логики  
- работа с мобильной платформой  

---

## 🎯 Цель разработки

Основные цели проекта:

- 📱 Изучение основ мобильной разработки  
- 🌐 Освоение HTML, CSS и JavaScript

## 🛠️ Используемые технологии

Проект разработан с использованием:

- **HTML** — структура приложения

## 🎮 Функционал приложения

### ✅ Реализовано:

- Игровое поле **3x3**
- Поочередные ходы игроков (❌ и ⭕)
- Определение победителя
- Проверка на ничью
- Возможность перезапуска игры

### 🚀 Планируется:

- Игра против компьютера  
- Подсчет статистики побед  
- Звуковые эффекты  
- Улучшение дизайна интерфейса

## 🚀 Запуск проекта

### 1. Установка зависимостей
npm install -g cordova

### 2. Создание проекта
cordova create tic-tac-toe
cd tic-tac-toe

### 3. Добавление платформы
cordova platform add android

### 4. Запуск приложения
cordova run android
