# Tic-Tac-Toe mobile app
Разработка мобильных приложений


О проекте

Данный проект представляет собой мобильное приложение — классическую игру «Крестики-нолики», реализованную с использованием веб-технологий и адаптированную под мобильные устройства с помощью Apache Cordova.

Основная идея проекта — разработать простое, понятное и доступное приложение, которое позволит пользователям играть в одну из самых известных логических игр прямо на своем смартфоне. Несмотря на кажущуюся простоту, проект охватывает базовые принципы разработки интерфейсов, обработки пользовательских действий и работы с мобильными платформами.

Цель разработки

Целью данного проекта является:
	•	Изучение основ разработки мобильных приложений
	•	Освоение технологий HTML, CSS и JavaScript
	•	Получение практического опыта работы с Apache Cordova
	•	Реализация игровой логики и пользовательского интерфейса
	•	Создание интуитивно понятного и удобного приложения

Используемые технологии

В процессе разработки используются следующие технологии:
	•	HTML — для создания структуры приложения
	•	CSS — для оформления и визуального стиля
	•	JavaScript — для реализации логики игры
	•	Apache Cordova — для сборки приложения и запуска на мобильных устройствах

Эти технологии позволяют создать кроссплатформенное приложение без необходимости писать нативный код для каждой платформы отдельно.

Описание функционала

Приложение включает в себя следующий базовый функционал:
	•	Игровое поле 3x3
	•	Возможность поочередного хода игроков (крестик и нолик)
	•	Определение победителя
	•	Проверка на ничью
	•	Перезапуск игры

В дальнейшем возможно расширение функционала, например:
	•	Добавление игры против компьютера
	•	Ведение статистики побед
	•	Добавление звуковых эффектов
	•	Улучшение дизайна интерфейса

Как запустить проект
	1.	Установить Node.js
	2.	Установить Cordova:
  npm install -g cordova
  3.	Создать проект:
  cordova create tic-tac-toe
  4.	Перейти в папку проекта:
  cd tic-tac-toe
  5.	Добавить платформу:
  cordova platform add android
  6.	Запустить приложение:
  cordova run android

Особенности проекта

Данный проект является учебным и направлен на освоение базовых навыков разработки. В процессе работы могут возникать сложности, связанные с настройкой среды, пониманием логики игры и взаимодействием технологий.

Тем не менее, именно через такие проекты формируется практический опыт и понимание того, как создаются реальные приложения.

Авторы
